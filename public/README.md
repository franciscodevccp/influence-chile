# public/

Coloca aquí los assets estáticos:

- `logo/logo.PNG` — logo de la agencia (Navbar e icono del sitio)
- **`logo-tiendas/`** — logos de clientes del portafolio (nombres esperados; ajusta en `lib/data.ts` si cambias extensión):
  - `domos.png`
  - `magic.jpeg`
  - `resin-epoxic.png`
  - `yole.png`
  - `yeet.png`
  - `onfayer.png`

Si falta un archivo o falla la carga, el portafolio muestra el nombre de la marca. En `lib/data.ts` revisa que el campo `instagram` sea el usuario real (sin `@`).
